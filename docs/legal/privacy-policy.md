## 2. 수집하는 개인정보 항목 및 수집 방법

### 2.1 회원가입 시 수집 항목
- **필수**: 이메일 주소, 비밀번호(해시 처리되어 저장, 원문 접근 불가)

### 2.2 서비스 이용 과정에서 수집되는 항목
- **아이(기록 대상자) 정보**: 이름 또는 애칭, 생년월일, 성별, 프로필 사진 — 이용자(보호자)가 직접 입력
- **사진 및 메타데이터**: 이용자가 업로드한 사진 파일, 촬영 시간(EXIF), 촬영 위치의 GPS 좌표(EXIF)
- **이용자 작성 콘텐츠**: 텍스트 일기, 마일스톤 기록
- **AI 생성 콘텐츠**: 업로드된 사진을 바탕으로 자동 생성된 일기 텍스트
- **가족 공유 정보**: 초대 코드, 공동 양육자 이메일(초대 수락 시)

### 2.3 자동으로 수집되는 항목
- 접속 로그, 접속 IP, 기기 식별 정보(OS 버전, 앱 버전)
- 푸시 알림 토큰 (알림 발송을 위한 기기 식별자)
- 서비스 이용 기록 (월별 업로드 횟수, 기능 사용 통계)

### 2.4 수집 방법
- 이용자가 앱 내에서 직접 입력
- 서비스 이용 과정에서 자동 수집
- 이용자가 업로드한 사진의 EXIF 메타데이터로부터 추출

## 3. 개인정보의 처리 목적

서비스는 수집한 개인정보를 다음 목적으로만 이용합니다.

1. **회원 가입 및 관리**: 본인 확인, 계정 관리, 부정 이용 방지
2. **서비스 제공**: 사진 저장, AI 일기 자동 생성, 타임라인 표시, 가족 공유
3. **AI 콘텐츠 생성**: 업로드된 사진과 메타데이터를 AI(Anthropic Claude) 모델에 전송하여 일기 생성
4. **알림 발송**: 일기 생성 완료, 마일스톤 달성, 기타 서비스 관련 공지
5. **서비스 개선**: 이용 통계 분석, 기능 개선, 오류 진단
6. **유료 서비스 의향 조사**: 이용자가 "Pro 출시 소식 받기" 버튼을 눌렀을 때 해당 의향 기록 (향후 서비스 기획용, 결제 정보 미수집)

## 10. 개인정보의 안전성 확보 조치

서비스는 개인정보의 안전성 확보를 위해 다음과 같은 조치를 취하고 있습니다.

1. **암호화**: 모든 통신은 HTTPS(TLS)로 암호화되며, 비밀번호는 단방향 해시(bcrypt 등)로 저장
2. **접근 통제**: 개인정보 접근 권한을 최소한의 관리자에게만 부여
3. **인증 토큰 보호**: 로그인 세션 토큰은 기기의 보안 저장소(iOS Keychain, Android Keystore)에 저장
4. **보안 업데이트**: 취약점 발견 시 신속히 패치 적용
5. **침해 대응**: 개인정보 유출 사고 발생 시 관련 법령에 따라 통지 및 조치

## 11. 자동 수집 장치의 설치·운영 및 거부

- 서비스는 웹 쿠키를 사용하지 않습니다.
- **푸시 알림 토큰**은 기기 OS에서 발급되며, 앱 내 알림 설정 화면 또는 기기 설정 > 알림에서 언제든 거부·철회할 수 있습니다.
- **위치 정보(GPS)**는 이용자가 업로드한 사진의 EXIF에서만 추출되며, 별도의 위치 추적을 수행하지 않습니다. 이용자는 사진 촬영 시 카메라 앱의 위치 저장 옵션을 끄거나, 업로드 전 사진 편집 도구로 EXIF를 제거하여 위치 수집을 차단할 수 있습니다.

## 기능 개요

타임라인 상단에 "N년 전 오늘" 메모리 카드를 표시해 Retention을 높인다.

## 구현 파일

| 파일 | 역할 |
|------|------|
| `src/components/MemoryCard.tsx` | 풀블리드 이미지 + 오버레이 텍스트 카드 컴포넌트 |
| `app/(tabs)/timeline.tsx` | entries에서 1~5년 전 ±2일 일기를 찾아 조건부 렌더링 |

## 핵심 동작

- entries 로드 후 오늘 날짜에서 1~5년 전을 순서대로 탐색
- 같은 달/일 ±2일 이내의 일기가 있으면 표시 (첫 번째 발견한 것만)
- 데이터 없으면 카드 숨김 (조건부)
- 탭 시 해당 일기 상세로 이동

## 알려진 한계

- 초기 유저는 1년치 데이터가 없으므로 노출 안 됨 (의도된 동작)
- ±2일 범위는 임의 설정 — 타이트하게 유지하는 편이 더 임팩트 있을 수 있음

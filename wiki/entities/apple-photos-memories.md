# Apple Photos — Memories & Apple Intelligence

## 정의
Apple의 사진 관리 서비스. iOS 18.1(2024년 10월)부터 Apple Intelligence 통합. Memory Movie Maker가 가장 진화된 서사 구조 영상을 자동 생성하는 기능.

## 핵심 AI 기능 (iOS 18 기준)

| 기능 | 설명 |
|------|------|
| Memory Movie Maker | 텍스트 프롬프트 입력 → AI가 전체 라이브러리 검색 → 챕터 구조 영상 자동 생성 |
| 기존 Memories | 자동 이벤트·인물·장소별 슬라이드쇼 생성 (프롬프트 불필요) |
| 얼굴·장소 인식 | 온디바이스 AI로 사람·장소 자동 태깅 |

## AI 텍스트 생성 수준
**없음.** Memory Movie는 "챕터 제목"이 있는 영상을 만들지만 문장 단위 일기 서사는 생성하지 않는다. 출력물은 영상(Video)만.

## 자동화 수준
**높음.** 기존 Memories는 100% 자동. Apple Intelligence Memory Movie는 텍스트 프롬프트 한 줄 → 완성.

## 프라이버시
**업계 최고.** 모든 AI 처리가 온디바이스. 사진이 애플 서버로 전송되지 않음. iCloud 사진은 E2E 암호화.

## 제약 사항
- iPhone 15 Pro / 16 시리즈 이상 필요 (구형 기기 사용 불가)
- 육아 마일스톤 개념 없음
- 텍스트 서사 미생성

## 가격
- 무료: 5GB iCloud
- iCloud+: $0.99/월 (50GB) ~ $59.99/월 (12TB)
- Apple Intelligence는 무료이나 지원 기기 제한

## 이음과의 비교
Apple Photos의 Memory Movie는 현재 시장에서 가장 아름다운 자동 영상 생성 기능. 그러나 **텍스트 서사 없고, 육아 맥락 없고, 일대기 형식 없음**. 이음은 "AI가 쓴 글 + AI가 만든 영상"의 조합으로 차별화.

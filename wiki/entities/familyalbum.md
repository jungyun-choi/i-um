# FamilyAlbum (みてね)

## 정의
MIXI, Inc.(일본)이 2015년 출시한 가족 사진 공유 앱. 한국·일본·아시아에서 가장 인기 있는 육아 앨범 앱. 2026년 2월 기준 2,900만 명 사용자.

## 핵심 기능
- 무제한 사진·영상 저장 (무료 = 압축, 유료 = 원본)
- 최대 100명 가족 공유, 날짜순 자동 정리
- Personal Pages: AI 얼굴 인식으로 아이별 피드 자동 분류
- **1s Movie**: AI가 자동으로 계절별 하이라이트 영상 생성
- 월 8장 무료 사진 인화
- AI 포토북 레이아웃 자동 제안

## AI 활용 현황

| AI 기능 | 수준 | 설명 |
|---------|------|------|
| 얼굴 인식 | ★★★★☆ | 딥러닝으로 아이별 사진 자동 분류 |
| 1s Movie 선택 | ★★★★☆ | 화질·반응수·얼굴 인식 기반 자동 클립 선택 |
| 포토북 레이아웃 | ★★★☆☆ | AI가 월별 베스트 사진 자동 배치 제안 |
| **텍스트 서사 생성** | **없음** | AI 일기/서술 기능 전무 |

## 자동화 수준
**매우 높음.** 사진 업로드만 하면 → 날짜 정리 → 아이별 분류 → 영상 생성 → 포토북 제안까지 자동.
단, 텍스트 일기는 수동 입력(Premium 전용).

## 평점 및 규모
- iOS: 4.8/5 · Google Play: 4.8/5
- 사용자 수: 2,900만 명 (2026년 2월)

## 강점
- 1s Movie — 감동적이고 자동화된 추억 영상 (핵심 차별점)
- 진짜 무료 (무제한 저장, 광고 없음)
- 할머니도 쉽게 쓰는 단순한 UI

## 약점
- **텍스트 서사/일기 AI 없음** — 모든 글은 부모가 직접 작성
- 이벤트·앨범별 분류 없음 (날짜순만 가능)
- UI가 2015년 수준으로 낙후
- 리액션 기능이 유료

## 이음과의 비교
FamilyAlbum이 **시각적 자동화(영상)** 에 강하다면, 이음은 **텍스트 서사 + 일대기 + 영상** 모두를 자동화하는 포지션.
FamilyAlbum의 1s Movie가 "감동의 영상"이라면, 이음은 "AI가 쓴 감동의 일기 + 영상".

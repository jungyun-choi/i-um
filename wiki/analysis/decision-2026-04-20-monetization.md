## (a) 가족 플랜 UX + Paywall 설계

### 현재 코드 분석
`apps/server/src/routes/photos.ts:21-25` — **월 30장 제한을 `user_id` 개인 기준**으로 체크.
즉 부부가 invite 코드로 join하면 자연스럽게 가구당 60장 가능 (사실상 "가족 플랜 비슷한 이득"을 무료로 제공 중).

### 핵심 결정: 한도 계산 기준 변경 필요

**현재 (개인 기준 30장):**
- ✅ invite 유인 강력 (한도 공유 = 2배 이득)
- ❌ "가족 플랜" 상품의 가치 명분 약함
- ❌ AI 비용 가구당 2배 부담

**제안 (아이 기준 30장 = family_members 합산):**
```ts
// 아이 기준으로 family 전체 업로드 수 합산
const { count } = await supabase
  .from('photos')
  .select('id', { count: 'exact', head: true })
  .eq('child_id', child_id)
  .gte('created_at', monthStart);
```
- ✅ 가족 플랜 상품 가치 명확 ("한도 공유 → 무제한으로")
- ✅ AI 비용 제어 가능
- ⚠️ 현재 무료 유저 중 2명 사용자에게 체감 한도 반감 — 베타 끝나고 P1 전환 시 적용 권장

### 가족 플랜 구조 (P1.5)
- **개인 플랜 4,900/월**: 1명, 무제한
- **가족 플랜 7,900/월**: 최대 N명 (부모+조부모), 무제한, 1개 결제로 전원 프리미엄
- 포지셔닝: "부부 각 3,950원, 한 사람이 결제하면 가족 전체 무제한"

### Paywall UX 변경 포인트
- 현재 `PaywallModal`이 이미 있으나 RevenueCat 미구현
- 30장 초과 시 모달에서:
  - "개인 플랜 4,900/월" vs "가족 플랜 7,900/월" 병렬 카드
  - 가족 플랜 추천 뱃지 (" 배우자도 함께 쓸 수 있어요")

## (b) AI 포토북 최소 기획

### 컨셉
**"1년 성장 에세이집"** — 다른 앱 포토북과의 차별점은 **AI 월간 레터 12편 + 타임라인 사진** 결합.

### 기술 스택 선택지

| 접근 | 장단점 | 복잡도 |
|------|--------|--------|
| 국내 포토북 API 제휴 (스냅스, 퍼블리스트) | 배송/인쇄 자동, 마진 20-30% | 중 |
| PDF 생성 → 사용자 직접 주문 | 초기 비용 최소, 수수료만 | 저 |
| 자체 인쇄 파트너 | 마진 40%+, 독점 디자인 | 고 |

**P2 진입 시 추천**: PDF 생성 MVP → 국내 API 제휴로 전환

### 선행 해결 과제

1. **원본 이미지 품질 문제 (치명적)**
   - 현재 `apps/mobile/app/upload.tsx:80-84` — 업로드 전 1024px resize
   - 인쇄 품질 기준 부족 (최소 2000~3000px 필요)
   - **대안**: 원본 별도 저장 + 압축본은 앱 표시용 분리
   - R2 스토리지 비용 증가 (장당 2-5MB × 수천 장) — 프리미엄 전용 기능으로 제한

2. **PDF 레이아웃 엔진**
   - `@react-pdf/renderer` (Node 서버) or puppeteer 기반 HTML→PDF
   - 한글 폰트 임베딩 (Pretendard 등)

3. **가격 구조**
   - 원가 (인쇄+배송): 약 15,000~20,000원
   - 판매가: 30,000~50,000원 (마진 40-60%)
   - AI 월간 레터가 있으니 "책 쓸 필요 없는 부모 에세이"로 프리미엄 포지션 가능

### P2 MVP 범위 (최소)
- 연말에 "올해의 성장 포토북" 타임라인 카드 노출
- AI가 자동 레이아웃 (월별 챕터 × 레터 + 사진 10장)
- 사용자는 미리보기만 → "주문하기" → 결제 → 배송
- 1개 포맷만 (A5 하드커버)

## (c) 경쟁사 가격 구조 deep dive

### 글로벌
| 앱 | 모델 | 가격 | 무료 범위 |
|----|------|------|----------|
| Tinybeans | Freemium | $5/월 ($48/년) | 무제한 업로드, 광고 포함 |
| FamilyAlbum (MIXI) | 무료 + 포토북 | 포토북 ¥1,500~3,000 | 완전 무료 (광고 없음) |
| 23snaps | 무료 | — | 완전 무료 |
| Glow Baby | Freemium | $9.99/월 ($59.99/년) | 제한된 트래킹만 |
| Apple/Google Photos Memories | 플랫폼 번들 | iCloud $0.99~/월 | 스토리지 연동 |

### 한국
| 앱 | 모델 | 가격 | 특이점 |
|----|------|------|--------|
| 쑥쑥찰칵 | 무료 + 포토북 | 포토북 단품 판매 | 일일 사진 자동 저장 |
| 베이비타임 | Freemium (약함) | 프리미엄 기능 연 50,000원 이내 | 구독 모델 약함 |
| 베이비빌리 | 무료 | — | 육아 커뮤니티 + 기록 |

### 핵심 인사이트

1. **한국 시장은 구독 저항 높음** — 일회성 포토북 구매가 더 자연스러움. 이음은 **구독+포토북 하이브리드**로 양쪽 다 커버.

2. **AI 서사 = 프리미엄 가격 정당화 근거** — 이음만 가진 기능이므로 4,900/월이 Tinybeans의 $5와 수평 비교되지만 **"AI가 써주는 일기"**로 프리미엄 포지션 가능.

3. **가족 플랜은 한국 OTT 모델 참고** — 웨이브/티빙 가족 요금제 7,900~9,900원대 익숙. 이음도 같은 가격대면 심리적 저항 낮음.

4. **포토북 시장은 FamilyAlbum이 증명** — 일본에서만 연 수백만 권 판매. 한국 부모도 감정 소비 성향 강함 (스냅스 연간 수백억 시장).

### 가격 책정 최종 권장

- **개인 프리미엄**: 4,900/월 (Tinybeans 대비 동급, 한국 체감 저렴)
- **가족 플랜**: 7,900/월 (OTT 가족요금 수준, 부부 각 ~4,000원)
- **연간 할인**: 49,000/년 (17% 할인 — Tinybeans 수준)
- **포토북**: 39,000원 (중간값, 연 1회 주문 기준 LTV +39,000)

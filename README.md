# 이음 (i-um)

> 아이의 모든 순간을 AI가 자동으로 기록해주는 제로에포트 육아 기록 앱

## 프로젝트 소개

부모가 직접 기록하지 않아도, AI가 사진·영상·메타데이터를 분석해 아이의 성장 일기와 하이라이트 영상을 자동으로 만들어줍니다.

## 핵심 기능

- **자동 일기 생성**: 사진 속 맥락을 AI가 분석해 감성적인 육아 일기 작성
- **스마트 앨범**: GPS, 시간, 인물 인식으로 자동 태깅 및 분류
- **성장 하이라이트**: 매월/매년 아이의 성장 영상 자동 편집
- **마일스톤 알림**: 첫걸음, 첫말 등 특별한 순간을 AI가 먼저 감지

## 기술 스택

- **Frontend**: React Native (iOS / Android)
- **Backend**: Node.js / Python
- **AI**: Vision AI, LLM (일기 생성), 영상 합성
- **Cloud**: AWS / GCP
